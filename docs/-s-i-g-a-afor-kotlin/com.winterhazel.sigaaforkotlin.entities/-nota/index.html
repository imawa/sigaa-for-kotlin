Representa uma nota cadastrada em uma disciplina.

Parameters

nota

Indica o valor da nota. Se for uma nota ainda não inserida (-- no SIGAA), o valor é -1.

notaMaxima

Indica a nota máxima. Se não possuir uma nota máxima definida, o valor é -1.

peso

Indica o peso da nota. Se a nota não possuir um peso definido, o valor é -1.
